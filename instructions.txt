Инструкция по использованию app.py

1. Подготовьте файл csv (в нашем случае с помощью прогноза риска сердечно сосудистого заболевания моделью main.ipynb)

2. Запустите программу

3. Откройте папку с проектом на вашем компьютере

4. Дважды кликните по папке venv, чтобы убедиться, что она есть (это специальная папка для работы программы)

5. Откройте терминал

6. В терминале перейдите в папку с проектом (cd "тут укажите путь к проекту")

7. Активируйте виртуальное окружение:
    Apply to main.ipynb
    activate

8. Запустите сайт командой: 
    Apply to main.ipynb
    8000

9. Если всё прошло успешно, появится сообщение, что сайт работает

10. Откройте браузер

11. Введите в адресной строке:
    localhost:8000

12. Откроется страница с формой для загрузки файла

13. Загрузите и посмотрите данные

14. Через несколько секунд вы увидите содержимое прогноза в виде списка JSON
